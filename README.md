# Intelligent Student Management System

An AI-powered platform for managing academic and administrative student activities.

## Features

- User authentication (Admin, Teacher, Student)
- Student management
- Attendance tracking
- Academic performance analysis
- Dashboard with statistics
- AI-based student performance prediction

## Tech Stack

Frontend
- React / Next.js
- Tailwind CSS

Backend
- Node.js
- Express.js

Database
- MySQL

AI Module
- Python

## Project Structure

frontend/ → User interface  
backend/ → REST API server  
database/ → SQL schema  
ai-service/ → AI prediction service  
docs/ → Architecture and documentation
